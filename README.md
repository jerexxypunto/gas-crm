# Client Managment System - GAS

### Features
- Genera un panel de control para cada ciente. []
    - Sistema de control de estado para cliente. []
    - Sistema de generacion de respuestas automaticamente para cada cliente. []
    - Sistema de control de Tareas por cliente. []
    - Sistema de control de eventos por cliente. []
- Centraliza el input de tareas accionables []
    - Ingreso de de Tareas multicanal: ( Directo, Whatspp, Gmail ) []
    - Ingreso de mensajes multicanal: ( Directo, Whatspp, Gmail ) []

### Sistema de control de estado para cliente
#### Descripción
Un Dashboard donde se pueda ver la información basica de cada cliente.
- Email
- Telefono
- Ubicacion
- Proyectos completados
- Proyectos en curso
- Proyectos pendientes
- Tareas ( ToDo List, tareas pequeñas desvinculadas a un proeycto )
- Eventos ( Videoconferencias, llamadas o reuniones precenciales )
- Numeros de cuenta

### Centraliza el input de tareas accionables
#### Descripción
Mediante un trigger se escuchara la entrada de correos o mensajes y se detectara las tareas de cada cliente.
